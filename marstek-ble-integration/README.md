This folder contains the assets needed for a PR to `home-assistant/brands` for the `marstek_ble` integration domain.

Steps to open the PR:

1. Fork `https://github.com/home-assistant/brands`.
2. Copy the `custom_integrations/marstek_ble/` directory from this folder into the fork (preserving the path).
3. Commit with a message such as `Add Marstek BLE brand assets`.
4. Open a PR against `home-assistant/brands`'s `master` branch referencing integration docs / repo.

The icons included here were sourced from the existing Marstek assets and meet the 512x512 PNG with transparent background requirements.
